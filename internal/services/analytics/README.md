# internal/services/analytics

**Layer:** AI Application Surfaces  
**Status:** 🔲 SLOT

Usage analytics and insights. Aggregates usage data across the fleet for admins and department leads.

## Planned responsibilities

- Per-user, per-dept, per-agent usage metrics (sessions, messages, tool calls, model usage)
- Feedback collection (thumbs up/down on responses)
- Performance metrics (response latency, tool error rates)
- Export for external dashboards (Grafana, Datadog)
- Powers the Analytics & Insights card in the enterprise UI

Build when: enterprise reporting is needed.
